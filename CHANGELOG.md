# Changelog

## 0.1.0

Initial MVP release.

- Single-screen mapping table UI
- Bidirectional OSC <-> MIDI message translation
- MIDI Note On/Off and CC support
- OSC with int32, float32, string arguments
- UDP and TCP transport for OSC
- MIDI device enumeration and selection
- Persistent settings and mappings (JSON)
- Real-time activity log
- Start/stop engine toggle
- Dark mode UI with Roboto Mono Variable typeface
- Cross-platform: macOS, Windows, Linux
